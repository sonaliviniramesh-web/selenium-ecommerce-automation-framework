# 🛒 E-commerce Automation Framework (In Progress)

This project demonstrates automation testing of an e-commerce web application using Selenium WebDriver.

## 🔧 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

## 📌 Features Implemented
- Login automation
- Basic framework setup using Page Object Model (POM)

## 🚀 Features In Progress
- Product search automation
- Add to cart functionality

## 🌐 Application Under Test
https://automationexercise.com

## 📁 Project Structure
- Base: Driver setup and configuration
- Pages: Page Object classes
- Tests: Test cases
